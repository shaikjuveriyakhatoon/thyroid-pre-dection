# Thyroid Prediction in Pregnancy and Normal Women
# Project Overview
Thyroid Prediction in Pregnancy and Normal Women is an IoT and web-based healthcare project designed to help identify the risk of thyroid disorders in both pregnant and non-pregnant women. The project aims to provide early screening and increase awareness, enabling timely medical consultation and reducing health complications associated with thyroid imbalances.
# Problem Statement
Thyroid disorders often remain undiagnosed due to a lack of awareness and delayed testing. During pregnancy, untreated thyroid conditions can affect both the mother and the baby's health. This project focuses on providing a simple and accessible solution for early thyroid risk prediction.
# Objectives
Predict the likelihood of thyroid disorders using patient health parameters.
Support early detection for pregnant and normal women.
Increase awareness about thyroid-related health issues.
Provide an easy-to-use web interface for users.
# Features
User-friendly web application.
Patient data input and management.
Thyroid risk prediction based on health parameters.
Secure storage of user information.
Responsive and accessible interface.
# Technologies Used
Frontend: HTML, CSS, JavaScript
Backend:  Node.js
Database: MongoDB
IoT Components: Sensors and microcontroller ,Esp32, DHT11,ECG.
# Applications

Healthcare awareness programs.

Educational and research purposes.
